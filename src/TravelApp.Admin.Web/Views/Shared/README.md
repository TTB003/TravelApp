# UI References for TravelApp

Thư mục này được dùng để lưu trữ các hình ảnh chụp màn hình (screenshots) từ ứng dụng **TravelApp.Mobile**.

### Hướng dẫn:
1. Chụp màn hình các trang: Explore, POI Detail, Scanner, và Library trên điện thoại/emulator.
2. Lưu vào thư mục này để làm tư liệu đối chiếu.
3. Khi yêu cầu tôi thiết kế, bạn hãy mô tả các đặc điểm quan trọng trong ảnh (ví dụ: "Nút Play có màu hồng #E31667, đổ bóng nhẹ, nằm chính giữa màn hình").
